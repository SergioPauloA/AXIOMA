# ⚡ AXIOMA — Urban Science Core

> **Hackeando a forma de aprender Ciências Exatas.**

AXIOMA é uma plataforma de estudos com uma proposta totalmente diferente do ensino tradicional. O projeto combina **gamificação**, **design Cyberpunk**, **Brutalism UI** e uma experiência altamente interativa para transformar conteúdos de Física, Química e Matemática em missões inspiradas em filmes, séries, animes, jogos e cultura pop.

---

## 📖 Sobre o Projeto

A proposta do AXIOMA é tornar o aprendizado muito mais envolvente.

Ao invés de apenas apresentar fórmulas e exercícios tradicionais, cada conteúdo é contextualizado dentro de universos conhecidos como:

- 🚗 Velozes e Furiosos
- ⭐ Star Wars
- 🧪 Breaking Bad
- ⚔️ The Witcher
- 🦸 Marvel
- 🎮 Cyberpunk 2077
- 🍥 Naruto
- 🏴‍☠️ One Piece
- 🧱 Minecraft
- ⚽ Champions League
- 🏀 NBA
- ...e dezenas de outros.

Cada exercício é apresentado como uma **missão**, incentivando o aluno a resolver problemas utilizando conceitos científicos em situações fictícias.

- Exercícios
- Gabaritos

---

# 🧠 Conceito

O projeto parte da ideia de que:

> "Aprender é muito mais eficiente quando existe contexto."

Ao utilizar referências da cultura pop, o estudante cria associações mais fortes com o conteúdo, facilitando memorização e entendimento.
